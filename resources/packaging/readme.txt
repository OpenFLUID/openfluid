Requirements
============

- boost libraries (version depends on the boost version used at build) 


Installation
============

It is advised to preferably install using automatic packages (deb, rpm, ...).
If you use archive (.zip, .tar.gz, ...), copy files to the right places by hand.


Usage
=====

Open a terminal and type the following command to get help 

   openfluid -h


Also refer to documentation available on the OpenFLUID web site:
https://www.umr-lisah.fr/openfluid/
